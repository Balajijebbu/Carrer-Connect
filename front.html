Front-End Developer

Company: Accenture

Location: Chennai, India

Average Salary: 9LPA

Description: Accenture is a leading technology company specializing in web development and software solutions.

As a Front-End Developer, you'll be responsible for:

  • Creating visually appealing and user-friendly web interfaces

    • Responsibilities:

    • Designing and implementing responsive web layouts using HTML and CSS
    • Developing interactive and dynamic web pages using JavaScript and front-end frameworks (e.g., React, Angular, or Vue.js)
    • Collaborating with designers and back-end developers to ensure seamless integration of front-end and back-end functionality
    • Optimizing web applications for performance and cross-browser compatibility
    • Creating and maintaining documentation for front-end code and user interfaces
    • Staying up-to-date with the latest front-end development trends and best practices

    Skills and Qualifications:

    • Bachelor's degree in Computer Science or a related field
    • Proficiency in front-end technologies such as HTML, CSS, JavaScript, and modern JavaScript frameworks (e.g., React, Angular, or Vue.js)
    • Experience with responsive web design and mobile-first development
    • Strong knowledge of web accessibility standards and best practices
    • Version control using Git
    • Experience with front-end build tools and package managers (e.g., Webpack, npm, or yarn)
    • Experience with RESTful APIs and asynchronous programming
    • Problem-solving skills and attention to detail
    • Excellent communication and teamwork abilities

Requirements:

  • Bachelor's degree in Computer Science or a related field
  • Proficiency in front-end technologies, including HTML, CSS, and JavaScript
  • Experience with front-end frameworks and libraries (e.g., React, Angular, or Vue.js)
  • Strong problem-solving skills and attention to detail
